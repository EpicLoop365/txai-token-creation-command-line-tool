Smart Token Features
Native Blockchain Capabilities
No smart contracts needed. All features are built into the chain.
⚖️

Mintable

Expand token supply as your project grows. Controlled by the issuer.

🔥

Burnable

Permanently reduce supply by burning tokens. Built-in deflation.

❄️

Freezable

Freeze individual addresses for compliance and security requirements.

✅

Whitelistable

Restrict transfers to pre-approved addresses only.

🔒

Clawback

Reclaim tokens from any holder. Essential for regulated assets.

🌐

IBC Enabled

Cross-chain transfers via Cosmos IBC. Interoperable by default.

⚠️

Global Freeze

Emergency pause on all transfers. Circuit breaker for your token.

🏛️

Governance

DAO-ready token governance. On-chain voting and proposals.

Quick Start
Up and Running in 60 Seconds
terminal
# Install the CLI
$ npm install -g @tx-agent/cli

# Set your API key
$ export ANTHROPIC_API_KEY=your_key

# Create a token with natural language
$ txai create "a gaming token called GEMS with 5M supply, mintable and burnable"

# Or use npx without installing
$ npx txai-create "a loyalty points token called POINTS with 1B supply"
